APIリファレンス

RESTful APIで外部システムとの連携が可能

スポット一覧取得

GET /api/spots/

全スポットの情報をJSON形式で取得

curl -X GET "https://your-app.com/api/spots/" \
-H "Authorization: Bearer YOUR_TOKEN"

スポット追加

POST /api/spots/add/

新しいスポットを登録

curl -X POST "https://your-app.com/api/spots/add/" \
-H "Authorization: Bearer YOUR_TOKEN" \
-d 'title=Tokyo Station&description=...'

スポット検索

GET /api/search/

キーワードによるスポット検索

curl "https://your-app.com/api/search/?q=tokyo"
